

### KORLOY 데모 앱 제작

* korloy 프로토타입에 대한 데모앱 제작 작업 진행 중.
* 현재 렛시 엔진상에는 코로이 product에서 제공하는 Data Matrix 타입의 barcode reader을 읽을수 없기 때문에 외부 모듈을 이용하였음.  
(엔진의 QRTracker는 zbar 사용) 
* 기본적인 스켈레톤 코드만 제작중이고 추후 flow나 목업이 나오면 구체적으로 구현할 예정.
